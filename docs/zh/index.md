# 时间经济项目

欢迎加入 **时间经济项目** ！

这是一个全新的经济制度，试图克服过去所有政治经济制度的缺点，并融合各[意识形态](3.意识形态.md)的优点。

时间经济项目目前还存在许多空白，如怎样设立法律法规、生产关系如何改造等问题，加入[讨论](#discussion)，发表你的见解！
